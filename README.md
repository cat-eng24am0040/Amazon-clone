# Amazon Clone

A full-stack Amazon-inspired e-commerce web application built to simulate a real online shopping platform. The project includes user authentication, product browsing, search functionality, shopping cart management, order processing, and secure database integration.
-->Features
1.User Registration & Login
2.Product Listing and Search
3.Shopping Cart Functionality
4.Product Details Page
5.Responsive UI Design
6.Database Integration
7.Order Management
8.Secure Authentication
-->Tech Stack
Frontend: HTML, CSS, JavaScript / React
Backend: Node.js, Express.js
Database: MongoDB
Version Control: Git & GitHub
