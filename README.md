# Amazon Clone 

A full-stack Amazon-inspired e-commerce web application built to simulate a real online shopping platform. The project includes user authentication, product browsing, search functionality, shopping cart management, order processing, and secure database integration.

---

##  Features

- User Registration & Login
- Product Listing and Search
- Shopping Cart Functionality
- Product Details Page
- Responsive UI Design
- Database Integration
- Order Management
- Secure Authentication

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript / React

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Version Control
- Git & GitHub

---

##  Team Contributions

- Frontend Development
- Backend API Development
- Database Management
- Deployment & Project Integration

---

##  Objective

The goal of this project is to understand full-stack web development, team collaboration using GitHub, and the implementation of core e-commerce functionalities similar to Amazon.

---

##  Learning Outcomes

- Full-Stack Web Development
- REST API Integration
- Database Design & Management
- Authentication & Security
- GitHub Collaboration Workflow
- Deployment Practices
